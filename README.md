# Amala

Video file sharing service.

## Tech Stack

- React 19
- Vite
- TypeScript
- Turborepo
- pnpm

## Getting Started

\`\`\`bash

# Install dependencies

pnpm install

# Start development server

pnpm dev

# Build for production

pnpm build

# Run linter

pnpm lint

# Type check

pnpm type-check
\`\`\`
